# Voice Chat App (Node + Express)

## Files
- `server.js` — Node server using Express
- `package.json` — dependencies + start script
- `public/index.html` — front-end

## Run Locally
1. Install dependencies:
   ```
   npm install
   ```

2. Start server:
   ```
   npm start
   ```

3. Open in browser:
   http://localhost:3000

## Notes
- This is a basic voice recognition demo.
- To use OpenAI, Twitch, or other features, we will add more code later.
